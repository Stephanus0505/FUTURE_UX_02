# FUTURE_UX_02
Smile Dental Care – Mobile Appointment Booking App

This project is a mobile appointment booking application designed as part of the UI/UX Design Internship (Task 2).

The goal of this app is to provide a smooth and user-friendly booking experience for dental patients. The design focuses on clarity, ease of navigation, and clear call-to-action elements to simplify the appointment scheduling process.

The mobile app includes four main screens:

Onboarding / Welcome Screen

Service Selection Screen

Date & Time Booking Screen

Appointment Confirmation Screen

The design maintains a clean visual system using soft blue tones to represent trust and professionalism. The layout is optimized for mobile interaction with clear tap areas and structured spacing.

Tool Used:

Figma (UI Design)
